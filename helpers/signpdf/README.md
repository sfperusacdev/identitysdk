# Document Signing Dependencies

To enable PDF document signing, the following dependencies are required:

## 🐍 Python Version

- **Python 3.12** (required)

## 📦 Python Packages

Install these with `pip`:

```bash
pyHanko==0.27.1
uharfbuzz==0.50.2
fonttools==4.58.0
pillow==11.2.1
poppler-utils==0.1.0
```
